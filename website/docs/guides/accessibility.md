# Доступность (Accessibility)

SimpleUI соответствует стандартам WCAG 2.1 для обеспечения доступности всех компонентов.

## Общие принципы

### Клавиатурная навигация

Все интерактивные элементы поддерживают навигацию с клавиатуры:

- `Tab` — переход между интерактивными элементами
- `Enter` / `Space` — активация элемента
- `Escape` — закрытие модальных окон и выпадающих меню

### ARIA атрибуты

Компоненты используют правильные ARIA атрибуты для описания состояния и свойств:

- `aria-label` — описание элемента
- `aria-disabled` — состояние отключения
- `aria-expanded` — состояние раскрытия
- `aria-haspopup` — наличие всплывающего меню

### Контраст цветов

Все цвета соответствуют стандартам WCAG AA для текста и AA для крупных компонентов:

- Обычный текст: минимум 4.5:1
- Крупный текст: минимум 3:1
- Графические элементы: минимум 3:1

## Компоненты и доступность

### Button

Кнопка поддерживает:
- Навигацию с клавиатуры
- Правильные ARIA атрибуты
- Фокус при навигации
- Индикатор отключения

```tsx
<Button disabled>Отключено</Button>
```

### ThemeProvider

ThemeProvider обеспечивает:
- Правильное переключение темы
- Сохранение предпочтений пользователя
- Поддержка системных тем

```tsx
<ThemeProvider theme="light">
  {/* Компоненты */}
</ThemeProvider>
```

## Рекомендации

### Для разработчиков

1. **Используйте семантические элементы** — используйте правильные HTML теги
2. **Добавляйте ARIA атрибуты** — описывайте состояние элементов
3. **Проверяйте контраст** — используйте инструменты для проверки
4. **Тестируйте с клавиатуры** — убедитесь в возможности навигации
5. **Используйте screen readers** — проверяйте чтение контента

### Инструменты для проверки

- [axe DevTools](https://www.deque.com/axe/) — расширение для Chrome
- [WAVE](https://wave.webaim.org/) — онлайн инструмент
- [Lighthouse](https://developers.google.com/web/tools/lighthouse/) — инструмент от Google
- [NVDA](https://www.nvaccess.org/) — screen reader для Windows
